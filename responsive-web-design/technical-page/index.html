html boilerplate

"but the title of the page is css quick guide, not html!" yeah yeah yeah, well you need this too, so might as well have it here:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8" /> <title>Hello, world!</title> <meta name="viewport" content="width=device-width,initial-scale=1" /> <meta name="description" content="" /> <link rel="stylesheet" type="text/css" href="style.css" /> <link rel="icon" href="favicon.png" /> </head> <body> <h1>Hello, world!</h1> </body> </html>

the stylesheet link connects our css file to our html file. the viewport meta lets us know the user's screen size so we can use media queries to set different layouts for different screen sizes

device width breakpoints

these are commonly used screen widths within @media queries for various devices:

  • 320px - 480px for mobile devices
  • 481px - 768px for ipads / tablets
  • 769px - 1024px for small screens / laptops
  • 1025px - 1200px for desktops / large screens
  • 1201px + for extra large screens / TV

for example, our css file for this site uses @media (min-width: 768px) to change the layout for desktop users

google fonts

to add google fonts, add <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=FAMILYNAME"> to your <head> element

to add multple fonts, use pipes to separate multiple FAMILYNAMEs

don't forget to put FAMILYNAME in quotes when you use it in your css code

variables

you declare variables in the :root selector so they can be accessed by all of the other selectors

use this syntax to declare a variable:

:root { --variable-name: value; }

and use this syntax to call the variable:

selector { property: var(--variable-name); }

as an example, this is how to give an element stripes in two variable colors:

:root { --first-color: #f93; --second-color: #f63; } #example { width: 50vw; height: 30vh; margin: 0.5rem 2rem; padding: 0.25rem; border-radius: 0.5rem; background: repeating-linear-gradient( var(--first-color), var(--first-color) 5%, var(--second-color) 5%, var(--second-color) 10% ); }
